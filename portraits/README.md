# portraits/

AI-generated 2D character portraits, one per species, derived from the
user's own photograph (image-to-image) so each bird keeps its real
plumage and shape while sharing one cohesive illustrated "house style".

- One file per species: `<species_zh>.webp` (transparent background preferred).
- Registered in `../data/portraits.json` as `{ "<species_zh>": "portraits/<species_zh>.webp" }`.
- The field guide's windowsill scene uses the portrait when present and
  falls back to the photo thumbnail otherwise — so this folder can fill in
  gradually, species by species.
- Generated by Agent-Bird's `bird-portrait-generator` skill.
