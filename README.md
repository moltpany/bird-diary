# Bird Diary — Field Guide

A lightweight, browser-based field guide and sighting diary for birdwatchers.

## Features

- Browse a curated field guide of common birds
- Log sightings with date, location, and notes
- Filter birds by family, habitat, or name
- Works offline — no backend required

## Project Structure

```
bird-diary/
├── index.html        # Main application entry point
├── css/
│   └── style.css     # Styles
├── js/
│   └── app.js        # Application logic
└── data/
    └── birds.json    # Field guide bird data
```

## Getting Started

Open `index.html` in any modern browser.  No build step or server required.
